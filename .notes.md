# GraphQL

## Create New User
```
mutation {
  addUser(input: { params: { username: "Bob", email: "bob@athens.com" } }) {
  	user {
    	username
    	email
    }
  }
}
```

## Edit User
```
mutation {
  editUser(input: { params: { username: "Bob Ward", email: "bob@athens.com" }}) {
    user {
      id
      username
      email
    }
  }
}
```
